# Image Assets

此資料夾用於存放網站圖片資源。

## 必要檔案（請替換）

### Favicon 相關
- `favicon-16x16.png` - 16x16 像素
- `favicon-32x32.png` - 32x32 像素
- `apple-touch-icon.png` - 180x180 像素

### Open Graph 圖片
- `og-image.png` - 建議 1200x630 像素，用於社群分享預覽

## Favicon 產生工具

推薦使用以下工具產生完整的 favicon 套件：
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

## 子資料夾

- `clients/` - 客戶 Logo
- `mockups/` - 產品截圖與示意圖
