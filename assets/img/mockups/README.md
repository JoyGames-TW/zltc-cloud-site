# Product Mockups

此資料夾用於存放正式官網使用的介紹示意圖（非示範截圖）。

## 現有檔案

- `pathology-system.svg` - 病理系統介紹示意圖
- `medical-dashboard.svg` - 醫療儀器數據儀表板介紹示意圖
- `custom-software.svg` - 企業軟體客製化介紹示意圖
- `article-featured-architecture.svg` - 資源頁主打文章示意圖
- `article-side-modernization.svg` - 資源頁側欄文章示意圖 A
- `article-side-maintenance.svg` - 資源頁側欄文章示意圖 B
- `article-side-frontend-backend.svg` - 資源頁側欄文章示意圖 C

## 建議規格

- 格式：SVG（可縮放且維持清晰）
- 內容：正式介紹用視覺，避免 UI 示範細節
- 注意：不得包含敏感資料（PHI、個資等）
