# Client Logos

此資料夾用於存放客戶 Logo 圖片。

## 預期檔案

- `kmuh.png` - 高雄醫學大學附設中和紀念醫院
- `vghtpe.png` - 臺北榮民總醫院
- `cmuh.png` - 中國醫藥大學附設醫院
- `chengda.png` - 正大醫院

## 建議規格

- 格式：PNG (透明背景) 或 SVG
- 尺寸：寬度 200-300px，高度依比例
- 用途：客戶背書牆、案例頁面

## 使用方式

目前網站使用文字徽章牆呈現客戶名稱。
如需使用 Logo，請將圖片放入此資料夾，並修改首頁 `index.html` 中的 client-wall 區塊。
