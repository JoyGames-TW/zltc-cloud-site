隱私權政策

本應用程式（以下簡稱「本服務」）由 湛盧科技有限公司 （英文名稱：Zhan-Lu Technology Co., Ltd.，以下簡稱「我們」）提供。

我們重視您的隱私權，並致力於保護使用者的個人資料與相關資訊。為了讓您瞭解本服務目前版本的資料處理方式，特訂定本隱私權政策。

一、適用範圍

本隱私權政策適用於您使用本服務時，與資料蒐集、處理及利用相關之事項。

二、目前上架版本的資料處理方式

本服務目前上架版本為純單機版本。

在目前上架版本中：

  • 不提供會員註冊或登入功能
  • 不提供雲端同步功能
  • 不提供對外後端 API 連線功能
  • 不會主動蒐集、儲存、傳輸或分享可識別您身分的個人資料

除非您所在裝置的作業系統、應用商店平台或您自行安裝的第三方服務另有其自身的資料處理機制，否則本服務目前版本本身不會主動向我們的伺服器傳送您的個人資料。

三、可能於裝置本機保存的資料

為使本服務能正常運作，部分資料可能僅儲存在您的裝置本機中，例如：

  • 遊戲或應用程式設定
  • 使用進度
  • 偏好設定
  • 單機存檔資料

前述資料僅作為本服務本機功能運作之用途，原則上不會上傳至我們的伺服器。

四、第三方服務與外部平台

本服務係透過應用商店平台提供下載與安裝。您在下載、安裝、付款、更新或使用應用商店相關服務時，該平台可能依其自身規範處理相關資料。此類資料處理行為並非由我們直接控制，建議您另行參閱該平台或相關服務提供者之隱私權政策。

五、兒童隱私

本服務目前版本未設計會員系統、社群互動或資料上傳功能，亦不會主動蒐集兒童個人資料。若您為法定代理人，且認為未成年人於使用本服務過程中有涉及個人資料疑慮，請透過本政策所載聯絡方式與我們聯繫。

六、資料安全

雖然目前上架版本原則上不會主動蒐集或傳輸您的個人資料，我們仍會盡合理努力維護本服務之安全性與穩定性，並於未來版本功能調整時，審慎評估相關資料保護措施。

七、未來功能調整與政策更新

若本服務未來版本新增以下功能：

  • 會員登入
  • 第三方登入
  • 雲端儲存或同步
  • 後端 API 連線
  • 使用分析
  • 錯誤回報
  • 廣告、推播或其他涉及資料蒐集之功能

我們將依實際功能內容更新本隱私權政策，並於更新後公布於本頁面或適當位置。
